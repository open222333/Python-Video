## 用法

```bash
usage: main.py [-h] [-p PATH] [-o OUTPUT]

optional arguments:
  -h, --help            show this help message and exit
  -p PATH, --path PATH  影片位置
  -o OUTPUT, --output OUTPUT 輸出位置
  -t TYPE, --type TYPE  轉換類型 av1, h264 預設all
  --onlymessage         只顯示訊息
  --convert_video       影片轉檔
  --convert_m3u8        轉檔m3u8
```
